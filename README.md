# Python Arcade Game System (Turtle Graphics)

## Overview

This project is a Python-based Gaming Arcade built using the Turtle Graphics module. It includes four classic games: Snake, Pong, Brick Breaker, and Flappy Bird. Each game runs independently and is accessed through a central menu-driven interface.

## Features

* Menu-based Arcade System
* Four games included:

  * Snake Game
  * Pong Game
  * Brick Breaker
  * Flappy Bird
* Built entirely using Python Turtle (no external libraries)
* Smooth animations and basic collision detection
* Keyboard-based controls

## Requirements

* Python 3.x
* Turtle module (pre-installed with Python)

## How to Run

1. Download all game files.
2. Run the main arcade file:

   ```bash
   python main.py
   ```
3. Choose a game from the menu.

## Games Summary

### Snake

Control a snake to eat food and grow while avoiding collisions.

### Pong

Two-player paddle game with bouncing ball physics.

### Brick Breaker

Bounce the ball to break bricks using a controllable paddle.

### Flappy Bird

Navigate a bird through moving pipes with gravity-based controls.
